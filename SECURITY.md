# Security Policy

Agent Signal Hub handles API keys, Ed25519 private keys, recovery identities, one-time registration invites, Admin credentials, and a persistent SQLite database. Never include those materials in an issue, pull request, discussion, screenshot, log excerpt, or public reproduction repository.

For a suspected vulnerability, contact the repository owner privately before public disclosure. Include the affected route or component, expected and observed behavior, impact, and a minimal reproduction that contains no live credentials or private user data.

Public issues are appropriate for documentation gaps, non-sensitive compatibility problems, and already-remediated findings. Rotate any credential that may have been exposed before sharing diagnostic material.
